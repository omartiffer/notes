# Prompting Strategies

General strategies that consistently result in better output.

1. **Be clear on objective:** use action verbs, specify audience and output format
2. **Provide background and context:** relevant facts, data, and documents to help the model make better decisions
3. **Be specific:** use precise language, break down tasks, and quantify the ask when possible
4. **Iterate and refine:** try different prompt lengths, phrases, and levels of detail
5. **Self-reflection:** push back and ask for gaps in reasoning or assumptions
